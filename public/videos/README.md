# Qui metti i tuoi MP4

Questa cartella contiene i video locali del corso "HTR Training".

## Convenzione di naming

Per coerenza con `content/course.json`, i file dovrebbero chiamarsi:

```
01-branding-parte-uno.mp4
02-branding-parte-due.mp4
03-case-study-updated.mp4
04-modulo-3-1.mp4
…
28-modulo-4-18.mp4
```

Puoi usare qualsiasi nome: aggiorna `videoPath` in `course.json` e tutto il
resto funziona. L'app cerca il file in `public/videos/<basename>`.

## Note legali

Inserisci solo file che hai scaricato ufficialmente e per cui hai diritto
d'uso. Questa webapp non effettua scraping, non accede a stream, non usa
cookie o token.
